このファイルは東工大ポータルのマトリクス認証を自動的にログインできるようになるpythonプログラムです。

使い方
1.matrix.xlsm のセルに対応する自分のマトリクスコードを入力します。
2.matrix.xlsm の学籍番号とパスワードの欄に自分の情報を入力します。(日本語の部分を消して自分のものにすり替えてください)
3.pythonファイルを実行します
